# 笨鸟先飞

《笨鸟先飞》是参考《Flappy Bird》而设计的，是从零开始自学游戏开发过程的第一款游戏，用于熟悉 Godot 游戏引擎和验证独立游戏开发技术、美术等方面可行性。
